## Description

Node.js API for NordHMS project

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```

## Migrations

Run migrations
```
 yarn typeorm:run-migrations
```

Revert last migration
```
 yarn typeorm:revert-migration
```

Generate migration based on Schema changes
```
 yarn typeorm:generate-migration src/database/migrations/<MigrationName>
```
